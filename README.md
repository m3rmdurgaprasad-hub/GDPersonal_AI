# AI Widget

An AI-powered widget built with **FastAPI (backend)** and **React (frontend)**.  
It supports:
- Text summarization (like Grammarly)
- Video analysis (placeholder for CNN/ViT models)
- Image generation (placeholder for diffusion models)
